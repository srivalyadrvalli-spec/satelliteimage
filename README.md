# Satellite Change Detection System

A Flask-based web application for detecting illegal land use changes using satellite images and AI.

## Features

- Change detection using before/after satellite images
- Visual change highlighting using OpenCV
- Land cover classification
- Detection of illegal activities like:
  - Illegal Construction
  - Lake Encroachment
  - Deforestation
- Notification system:
  - Web alerts
  - Email alerts
  - SMS alerts using Twilio

## Installation

1. Clone the repository
2. Install dependencies

pip install -r requirements.txt

3. Run the application

python app.py

4. Open in browser

http://127.0.0.1:5000

## Technologies Used

- Python
- Flask
- OpenCV
- NumPy
- Twilio API
- HTML / CSS

## Purpose

This project detects illegal land use changes using satellite images to help authorities monitor environmental violations.
